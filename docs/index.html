WHAT IS THIS

Mission-WM builds model-only game engines: neural networks that generate every frame of a playable game from pixels, player input and their own memory — no simulator in the loop at inference time.

These two games are our data engines. Every state is a serializable dict, every run is bit-deterministic, and any moment can be forked into counterfactual branches — the same keys, a different world. That forking supervision (branch-consistency), an event-memory ledger (lives / ammo / kills, RAM-supervised in training, self-predicted at inference) and a legal-action red-team metric (BreakLength) are the research agenda. The neural twins of both games are training now; when they land, you'll play them here, side by side with these originals.
